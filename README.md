# EDAware App

Working MVP application repo for EDAware, a hospital and EMS coordination platform.

## Current MVP

This first version is a lightweight browser app with:

- EMS intake form
- alert creation and tracking
- incoming patient board
- admin configuration for alert templates, routing, and roles
- local persistence using browser storage

## How to run

Open `index.html` in a browser.

No build step is required for this first MVP.

## Why this approach

This keeps the first version immediately runnable while the product shape is still evolving. It can later be migrated into a framework-based application without losing the core workflow decisions.
